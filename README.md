# 📚 VerseVibe - Book Recommendation System

##  Overview

VerseVibe is an AI-powered Book Recommendation System that suggests books to users based on their interests and behavior. It uses both **Collaborative Filtering** and **Popularity-Based Recommendation** techniques to deliver personalized suggestions.

---

##  Features

*  Personalized Book Recommendations
*  Popular Books Suggestion
*  Collaborative Filtering Model
*  Fast Predictions using Pre-trained Models
*  Data-driven approach

---

##  Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Pickle (Model Storage)
* Streamlit (optional UI)

---

##  Dataset

The dataset used in this project includes:

* Books dataset
* Users dataset
* Ratings dataset

 Source: Public datasets (e.g., Kaggle Book Recommendation Dataset)

---

## ⚙️ How It Works

1. Data preprocessing and cleaning
2. User-item matrix creation
3. Similarity calculation
4. Recommendation generation
5. Model saved using Pickle
